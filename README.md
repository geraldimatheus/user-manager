# 🚀 Gerenciador de Usuários

Aplicação web desenvolvida com HTML, CSS e JavaScript puro para gerenciamento dinâmico de usuários em uma lista.

---

## 🔗 Demonstração

(Adicione aqui depois o GitHub Pages, se quiser publicar)

---

## 📌 Funcionalidades

- ➕ Adição de usuários à lista
- 🚫 Validação de campo vazio
- 🔁 Prevenção de usuários duplicados
- ❌ Remoção de usuários individualmente
- ⌨️ Suporte à tecla Enter para envio
- 🎨 Interface moderna com layout responsivo

---

## 👨‍💻 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de praticar:

- Manipulação de DOM
- Eventos em JavaScript
- Estruturação de layout com Flexbox
- Boas práticas de organização de código

A proposta simula um pequeno sistema de gerenciamento, semelhante a interfaces reais utilizadas em ambientes corporativos.

---

## 🛠️ Tecnologias Utilizadas

- HTML5 → Estrutura da aplicação
- CSS3 → Estilização e layout (Flexbox)
- JavaScript → Lógica e interatividade
